    1. Define scoring weight constants at module level:
       ```
       WEIGHT_CAPITAL_EFFICIENCY = 0.45
       WEIGHT_VOLATILITY = 0.35
       WEIGHT_FUNDAMENTALS = 0.20
       ```

    2. Implement `compute_wheel_score(stock: ScreenedStock, all_passing_stocks: list[ScreenedStock]) -> float`:
       - Capital efficiency (0-1): Normalize price across all passing stocks. Lower price = higher score. When min_price == max_price, use 0.5.
         `cap_eff = 1.0 - (stock.price - min_p) / (max_p - min_p)`
       - Volatility proxy (0-1): Normalize hv_30 across all passing stocks. Higher HV = higher score. When hv_30 is None, use 0.5. When min_hv == max_hv, use 0.5.
         `vol_score = (stock.hv_30 - min_hv) / (max_hv - min_hv)`
       - Fundamental strength (0-1): Average of up to 3 sub-components:
         - Net margin: `min(stock.net_margin / 30.0, 1.0)` (30%+ = 1.0)
         - Sales growth: `min(max(stock.sales_growth, 0) / 30.0, 1.0)` (30%+ = 1.0)
         - Debt/equity: `max(1.0 - stock.debt_equity, 0.0)` (0 debt = 1.0, 1.0+ debt = 0.0)
         - If no components available, use 0.5.
       - Combine: `raw = WEIGHT_CAPITAL_EFFICIENCY * cap_eff + WEIGHT_VOLATILITY * vol_score + WEIGHT_FUNDAMENTALS * fund_score`
       - Return: `round(raw * 100, 2)`

    3. Add tests to tests/test_pipeline.py. Create helper function to build lists of ScreenedStock with specific field values for normalization testing. Test edge cases: single stock, all identical prices, None fields.

    Use `import logging as stdlib_logging` per project convention.
